# Kata PHP

## Docker

- lancer le container :
```docker-compose up -d```

- Se connecter le container :
```docker-compose exec eurelis bash```

- Tester tous les exercices (une fois connecté au container) :
```php check.php```

- Tester l'exercice 02 (une fois connecté au container) :
  ```php check.php 02```
